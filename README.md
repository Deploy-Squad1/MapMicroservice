# MapMicroservice

Server-side application for the interactive map feature.

## Features

- Artifact management (CRUD operations)
- JWT-based Role-Based Access Control (RBAC)
- Automated database migrations

Built with Go and PostgreSQL/PostGIS.

## Setup Project

Create a `.env` file in the root directory with your database and JWT credentials:

```
JWT_SECRET=your_secret_key
DB_USER=postgres
DB_PASSWORD=admin
DB_HOST=localhost
DB_PORT=5432
DB_NAME=core_db
```

Install dependencies and run the development server:

```
go mod download
go run ./cmd/api/main.go
```

API runs on: http://localhost:8080

## Local Development (Docker)

Build and run the container:

```
docker build -t map-service .
docker run -d -p 8080:8080 --env-file .env --name map-app map-service
```

Stop container:

```
docker rm -f map-app
```

## API Connection

The frontend communicates with this microservice. Base API endpoints are located at: http://localhost:8080/api/artifacts.

Authentication requires an `access_token` cookie generated by the CoreMicroservice.
